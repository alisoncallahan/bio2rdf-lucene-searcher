Search Bio2RDF Lucene Indexes

Example queries

To search all namespaces in Bio2RDF, and retrieve results as XML (the default content-type) or JSON:

  • query=cell
  • query=cell&format=json

To search all namespaces in Bio2RDF, and retrieve results as XML (the default) or JSON, but return the text content-type:

  • query=cell&asText=true
  • query=cell&format=json&asText=true

To restrict your search to a specific namespace:

  • query=cell&ns=go
  • query=cell&ns=go&asText=true
  • query=cell&ns=go&format=json
  • query=cell&ns=go&format=json&asText=true

Understanding search results

Each search result consists of the URI of the entity matching the input search term, its label (rdfs:label), its type (rdf:type) and, if found, its definition (obo:definition).

If the search was not restricted by namespace, the results also include the number of hits for the 5 namespaces with the most search results.

Specifically, XML results will take the following form:

<results>
   <bio2rdfHits>
      <result>
         <uri>URI of result 1</uri>
         <name>Name of result 1</name>
         <type>Type of result 1</type>
         <definition>Definition of result 1</definition>
      </result>
      <result>
         <uri>URI of result 2</uri>
         <name>Name of result 2</name>
         <type>Type of result 2</type>
         <definition>Definition of result 2</definition>
      </result>
   </bio2rdfHits>
   <nsHits>
      <nsHit>
         <ns>Name of source</ns>
         <prefix>Namespace prefix</prefix>
         <count>Number of search hits</count>
      </nsHit>
   </nsHits>
</results>
